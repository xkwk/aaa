# Streamlit in Snowflake App

`app.py` is an output/review artifact. It is not the orchestration interface.
Users still operate the workflow by talking to GitHub Copilot in VS Code.

Deploy this app in Snowflake after the `ACTUARIAL_AI_DEV.APP.DASHBOARD_SUMMARY`
and `ACTUARIAL_AI_DEV.RUNS.VALIDATION_RESULTS` tables are populated.
